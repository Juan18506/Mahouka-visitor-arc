## Installation and usage

### Runing the project

Clone this repository: 

```
git clone https://github.com/juan18506/mahouka-visitor-arc.git
cd mahouka-visitor-arc
```

Install packages :

```
pnpm install
```

When installation is complete, run :

```
pnpm run dev
```
